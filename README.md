## 💻 Run Locally

Clone the repository:

```bash
git clone https://github.com/Harika1234-git/HARIKA-portfolio.git
```

Go to the project folder:

```bash
cd HARIKA-portfolio
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Open your browser:

```text
http://localhost:5173
```
